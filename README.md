# Savant Website

A beautiful, modern website for Savant - an app that helps autistic people with attention deficit differences focus on their special interests.

## Features

- **Dark Blue Gradient Background** - Beautiful gradient from deep navy to bright blue
- **Modern Design** - Clean, professional aesthetic with smooth animations
- **Responsive Layout** - Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations** - Floating shapes, hover effects, and scroll animations
- **Accessible Navigation** - Fixed navbar with smooth scrolling to sections

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive JavaScript functionality

## How to Use

Simply open `index.html` in your web browser to view the website. No build process or server required!

For local development, you can use a simple HTTP server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## Design Highlights

- **Color Scheme**: Deep navy (#0f172a) to bright blue (#1e40af) gradient
- **Typography**: Inter font family for modern, readable text
- **Glass Morphism**: Frosted glass effect on cards with backdrop blur
- **Animations**: Floating background shapes, hover effects, and scroll-triggered animations
- **Accessibility**: Smooth scrolling, proper semantic HTML, and responsive design
